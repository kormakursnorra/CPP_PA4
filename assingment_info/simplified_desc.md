# PA4 Requirements

## Must Have
- [ ] Uses inheritance, templates, OR variants (at least one)
- [ ] Makefile included
- [ ] README with two sections: one for players, one for programmers/code design

## Core (50%)
- [ ] Two entities can duel (turn-based)
- [ ] Each entity has stats (Health, Attack, Defence minimum)
- [ ] Damage mechanic implemented (formula clearly described in README)
- [ ] Each turn: player chooses an action (e.g. Attack, Flee)
- [ ] Game ends when one entity reaches 0 HP
- [ ] Multiple entity types with different behaviour (via polymorphism)
- [ ] Player can select which entities duel (CLI args or menu)
- [ ] Some pseudo-randomness in gameplay

## Additions (Grade Boosts — pick any)
- [ ] Teams of multiple entities
- [ ] Special abilities per entity type
- [ ] More stats (mana, energy, etc.)
- [ ] Damage types & resistances
- [ ] ASCII art / richer UI
- [ ] Overworld map with random encounters
- [ ] Save / load game
- [ ] Levelling system
- [ ] Equipment & items
- [ ] Formations (positional gameplay)
- [ ] Boss encounter / phases
